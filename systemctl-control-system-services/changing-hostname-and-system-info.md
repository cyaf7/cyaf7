# Changing Hostname & System Info

**Set hostname:**

{% hint style="info" %}
hostnamectl set-hostname \<newhostname>
{% endhint %}

View hostname:

{% hint style="info" %}
cat /etc/hostname
{% endhint %}

System info:

{% hint style="info" %}
cat /etc/os-releasename
{% endhint %}

uname -a

dmidecode -> for hardware info

arch -> check if system is 32/64-bit
