# wc — Word Count

Counts things in a file or text — lines, words, or characters.

Shows all: lines, words, and bytes:

{% hint style="info" %}
wc file.txt
{% endhint %}

shows only lines:

{% hint style="info" %}
wc -l file.txt
{% endhint %}
